# 🚗 Second Hand Tyres Store

This is a simple e-commerce website for buying and selling second hand tyres.

## 🌐 Live Website
https://abhirana1318.github.io/second-hand-tyres/

## 🛒 Features
- View available tyres
- Price display
- Simple and clean design
- WhatsApp order system

## 📸 Products
- Used Tyre 14 inch – ₹1200
- Used Tyre 15 inch – ₹1500
- Used Tyre 17 inch – ₹2000

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📞 Contact
For orders, contact on WhatsApp.
